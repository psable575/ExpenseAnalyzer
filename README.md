# FinanceFlow 💸 (Expense Analyzer v2)

FinanceFlow is a modern, premium web application built for hackathons to help users track their transactions, set custom category budgets, and manage long-term financial savings goals. It features a stunning "Web3/Fintech" dark-mode aesthetic with ambient glowing backgrounds and glassy UI components.

## ✨ Key Features

- **🔒 Secure Authentication Flow:** Personalized dashboard entry via a beautifully designed login gateway.
- **📊 Smart Analytics Dashboard:** Instantly view monthly income, net expenses, and your total balance.
- **💡 AI-Style Insights Engine:** A custom logic engine that scans your transactions against your strict budgets and dynamically warns you if you are overspending in categories (e.g., Subscriptions or Dining out).
- **📝 Dynamic Transaction Ledger:** A smart input log that separates "Income" streams (Salary, Freelance, Investments) from "Expenses" (Rent, Food, Shopping).
- **🎯 Financial Goals Tracker:** Set custom targets (like buying a laptop or building an emergency fund) and visually track your funding progress.
- **📈 Advanced Data Visualization:** Built-in interactive Pie Charts and Bar Graphs comparing your actual spending against your planned budget limits using Recharts.

## 🛠️ Tech Stack

- **Frontend Framework:** React.js powered by Vite for lightning-fast hot module replacement.
- **Styling:** Advanced Vanilla CSS implementing Modern Glassmorphism & Animated Mesh gradients.
- **Icons:** `lucide-react` for crisp, professional SVG iconography.
- **Data Visualization:** `recharts` for highly responsive, interactive charting.
- **Data Persistence:** React Context API paired with native browser `localStorage` ensuring lightning-fast performance without the need for a backend SQL database setup. Perfect for reliable Hackathon demos.

## 🚀 How to Run Locally

Since this app uses local storage for the database, setting it up is incredibly easy and requires zero backend configuration.

1. **Navigate to the directory**:
   ```bash
   cd expense-analyzer
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the app**:
   Open Your Browser and go to the Local url provided in the terminal (usually `http://localhost:5173/` or `http://localhost:5174/`).

## 🎨 Design Philosophy
This app was engineered to stand out during a Hackathon pitch. Moving away from standard flat-design, FinanceFlow implements deep Z-axis layering. The app brings standard dark-mode to life using massive, deeply saturated, blurred colorful orbs floating behind sheer, frosted-glass transaction panels to replicate the feeling of a million-dollar Silicon Valley startup.
